Projects

Smithy

Welcome, noble Knight. Enter the realm and craft for the kindom's finest warriors! This is a medieval themed crafting game inspired by Little Alchemy. Drag and drop items, find new materials, and craft to your heart's content.

ResearcherRouter

A Discord bot that helps researchers find people who discussed specific topics by automatically indexing research papers and discussions using Qdrant vector database. Turns a Discord channel into a searchable index of research topics linked to knowledgeable members. Uses a vector store with metadata, forum migration for data ingestion, and a bot interface to connect users with the right experts via Discord.

SignalRank

Bot solution for Discord which utilizes Python and Generative AI integration to deliver instant knowledge within a gamified environment. Built for the UF IEEE Signal Processing Society Discord server to enhance community engagement and support other community-based data projects.

Cooks' Match

An automatic recipe generator created with the EasyOCR Python module. Upload the image to the app and extract the text, then automatically generate the recipe using Google Gemini. Project developed for MiniHacks 2025.
